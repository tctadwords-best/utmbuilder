คำอธิบายโครงสร้าง UTM ของ Google (ย่อความสั้นๆ)

Source (แหล่งที่มา): ระบุชื่อโซเชียลมีเดียหรือเครื่องมือ เช่น facebook, line, tiktok
Medium (สื่อ): ประเภทสื่อโฆษณา เช่น social, paid_ad, affiliate
Campaign (ชื่อแคมเปญ): ตั้งชื่อโครงการหรือแคมเปญโปรโมตเพื่อรวมกลุ่ม เช่น brand_day_99
Term (คีย์เวิร์ด): มักใช้กับการยิงแอดประเภทค้นหา (Search Ads)
Content (รูปแบบโพสต์): ใช้ระบุชนิดการดีไซน์ชิ้นงานหรือเนื้อหา เช่น video_v1

ตรวจสอบความถูกต้อง นำลิงก์ UTM และลิงก์สั้นเดิมกลับมาใช้ใหม่ หรือตรวจวิเคราะห์จำนวนการคลิก

รวมประวัติลิงก์ทั้งหมด (0 รายการ) ✓ ย่อด้วยระบบความปลอดภัยสูง (ลิงก์ทุกตัวใช้งานได้จริงแบบถาวร)
วันที่สร้าง ช่องทาง ลิงก์ดั้งเดิม & ลิงก์สั้น ลิงก์ UTM เต็ม คลิก (จำลอง) จัดการ

ตั้งค่าระบบ & เชื่อมต่อ API ลิงก์ย่อภายนอก (ทางเลือกเพิ่มเติม)

กำหนดสิทธิ์และกุญแจ Access Token เพื่อสลับกลับไปใช้บริการของ Bitly

เชื่อมต่อบัญชี Bitly API (Optional)

* ปกติระบบเปิดใช้งานตัวย่อลิงก์ฟรีจาก **is.gd** (Local fast engine) ให้คุณเรียบร้อยแล้วโดยไม่ต้องทำอะไรเพิ่ม แต่หากบริษัทของคุณมี Account แบรนด์ของตนเองบน **Bitly** ก็สามารถนำ Token มาใส่ในฟิล่างเพื่อเปลี่ยนระบบมาใช้แบรนด์ของคุณเองได้ครับ

* ระบบจะจัดเก็บกุญแจความปลอดภัยนี้ไว้ในเครื่องคอมพิวเตอร์ของคุณเท่านั้น (จะไม่ถูกส่งไปเซิร์ฟเวอร์อื่นใดนอก bitly.com โดยตรงเพื่อย่อลิงก์)
